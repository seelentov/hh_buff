# HH_BUFF — Система сбора и анализа данных

Проект предназначен для автоматизированного мониторинга, сбора данных по поисковым запросам и их последующей визуализации. Система включает в себя REST-сервер для управления параметрами сбора и интеграцию с дашбордами Grafana.

## Основные ссылки

* **REST API сервер:** http://185.104.114.242:8701
* **Документация API (Swagger):** http://185.104.114.242:8701/swagger/index.html
* **Публичный дашборд (Grafana):** [Перейти к статистике](http://185.104.114.242:3000/public-dashboards/080945213e28475e8640165e7a71f462)

## Функционал

### 1. Сбор данных
Система позволяет добавлять собственные поисковые запросы через API. После добавления запроса система начинает аккумулировать данные для последующего анализа.

### 2. Просмотр данных
Просмотр собранной информации осуществляется через эндпоинт:
`GET http://185.104.114.242:8701/rest/data`

Доступные параметры запроса:
* `start_date` — фильтрация по начальной дате.
* `end_date` — фильтрация по конечной дате.
* `queries` — фильтрация по конкретным поисковым запросам.

### 3. Визуализация
Статистика по собранным данным автоматически выводится на публичный дашборд Grafana. Это позволяет отслеживать динамику и ключевые показатели в реальном времени.

## Технические подробности

* **Порт REST-сервера:** 8701
* **Технологический стек:** Go (Golang), Gin, MongoDB/SQLite, Swaggo, Grafana.
* **Интерфейс API:** Описан в формате OpenAPI (Swagger).

## Установка и запуск

Для локального запуска проекта (при наличии исходного кода):
1. Убедитесь, что установлена среда Go.
2. Скачайте зависимости: `go mod tidy`.
3. 1. Запустите rest сервер: `go run cmd/rest/main.go`.
3. 1. Запустите парсер: `go run cmd/daemon/main.go`.
